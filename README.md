## 📋 About the Event

**HACK::MET 2026** is a 48-hour non-stop student innovation hackathon organized by **E-Cell MET, Institute of Technology (Polytechnic)**, MET Bhujbal Knowledge City, Adgaon, Nashik.

| Detail | Info |
|--------|------|
| 📅 Date | 12–13 July 2026 |
| 📍 Venue | MET Institute of Engineering, Nashik |
| ⏱ Duration | 48 Hours (Non-Stop) |
| 👥 Team Size | 2 – 4 Members |
| 🎓 Eligibility | UG / PG Students (Any Stream) |
| 💰 Registration Fee | ₹299 / Team |
| 📤 Registration Deadline | 20 June 2026 |

## 🌟 Website Features

- ⏱ **Live Countdown Timer** — counts down to hackathon start (12 July 2026, 09:00 AM IST)
- 💻 **Terminal-style UI** — hacker aesthetic with dark mode, green-on-black palette
- 📱 **Fully Responsive** — optimized for mobile, tablet, and desktop
- 🎯 **6 Track Cards** — with domain tags and hover effects
- 🏆 **Prize Section** — with podium layout and special awards
- 📅 **Interactive Timeline** — with live/upcoming badges
- 👨‍⚖️ **Jury Section** — avatar cards for all judges
- 💼 **Sponsors Grid** — tiered sponsorship display
- ❓ **FAQ Accordion** — expandable Q&A section
- 📝 **Registration Form** — with animated submit button feedback
- 🎞 **Marquee Ticker** — scrolling event info banner

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/rohansachinpatil/hackmet-2026.git

# Navigate to the project
cd hackmet-2026

# Open directly in browser
open index.html
# OR on Windows:
start index.html
```

No npm install. No build process. It just works. ✅
